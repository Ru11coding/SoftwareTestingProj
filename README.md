📌 Manual Testing Project – Test Plan & Test Cases

This repository contains a structured manual testing project focused on creating and documenting a comprehensive Test Plan and detailed Test Cases based on application requirements.

🔍 Project Overview

The objective of this project is to demonstrate the core concepts of Software Testing Life Cycle (STLC) by designing effective test strategies and validating application functionality through well-written test cases.

📂 Project Contents

* ✅ Test Plan Document

  * Scope of testing
  * Objectives
  * Testing strategy
  * Resources and timelines
  * Entry & exit criteria

* ✅ Test Cases Document

  * Detailed test scenarios
  * Step-by-step execution steps
  * Expected vs Actual results
  * Test data and priority

🧪 Testing Approach

* Functional Testing
* Positive & Negative Testing
* Boundary Value Analysis
* Equivalence Partitioning

🛠 Tools Used

* Microsoft Excel / Google Sheets
* Basic documentation tools

🎯 Key Features

* Clear and easy-to-understand documentation
* Real-world testing approach
* Well-structured test cases for better traceability
* Beginner-friendly project for QA learning

🚀 Purpose

This project is designed to:

* Practice writing Test Plans and Test Cases
* Build a strong foundation in Manual Testing
* Showcase QA skills to recruiters and hiring managers
